# E-commerce Frontend

Онлайн дэлгүүрийн frontend хэсэг.

## Технологи

- Next.js 14
- TypeScript
- Tailwind CSS
- React
- Zustand (State Management)
- Axios (API calls)

## Суулгах

```bash
yarn install
```

## Ажиллуулах

Development:
```bash
yarn dev
```

Production build:
```bash
yarn build
yarn start
```

## Тохиргоо

API URL-ийг environment variable-аар тохируулна:
- `NEXT_PUBLIC_API_URL` - Backend API URL (default: http://localhost:5001/api)
